# Browser Integration

SupaSidebar integrates seamlessly with 14 major web browsers through AppleScript - no extensions needed, just powerful keyboard shortcuts and intelligent automation.

## When to Use

- **Multi-browser workflows**: Work across Safari, Chrome, Firefox, Arc, and more without switching tools
- **Quick page capture**: Save interesting pages while researching without breaking flow
- **Cross-browser management**: Track tabs and activity across all your browsers simultaneously
- **Context-based browsing**: Open work links in Chrome, personal in Safari, dev in Firefox
- **Extension-free integration**: Enjoy deep browser integration without installing browser extensions
- **Keyboard-driven efficiency**: Control everything with global shortcuts

## How It Works

SupaSidebar uses AppleScript to communicate with browsers, enabling seamless integration without browser extensions:
- **Smart Save & Copy**: Context-aware shortcuts that work across all browsers
- **14 Browsers Supported**: Safari, Chrome, Firefox, Edge, Arc, Brave, and more
- **Browser Selection**: Choose which browser opens each link
- **Live Tabs**: Real-time tab tracking across supported browsers
- **No Configuration**: Works immediately with Automation permissions

## Core Browser Features

### 1. Smart Shortcuts

**Context-aware keyboard shortcuts that adapt to your active application**

Press the same shortcuts in any browser or Finder - SupaSidebar automatically does the right thing:
- **⌘⌃S (Smart Save)**: Save browser pages OR add Finder items
- **⌘⌃C (Smart Copy)**: Copy URLs OR file paths

[PLACEHOLDER: Screenshot showing Smart Save in browser, then Finder]

**Works with:**
- All 14 supported browsers
- Finder (files and folders)
- No configuration needed

**Learn more:** [Smart Shortcuts →](smartShortcuts)

### 2. Browser Support

**14 supported browsers with two levels of integration**

SupaSidebar works with Safari, Chrome, Firefox, Edge, Arc, Brave, Vivaldi, Comet, Orion, Zen, Atlas, Wavebox, Dia, and Helium:

**Basic Integration (All 14):**
- Smart Save and Smart Copy
- Browser detection
- Link opening

**Live Tabs (10 browsers):**
- Real-time tab tracking
- Tab switching
- Tab search

### 3. Browser Selection

**Choose which browser opens your saved links**

Control exactly which browser opens each link with flexible options:
- **Per-link assignment**: Set once, use forever
- **Active browser mode**: Opens in current browser dynamically
- **Always ask mode**: Choose every time
- **One-time selection**: ⌥Click or ⌥Return for ad-hoc choices

- Work links in Chrome, personal in Safari

## Permissions

### Required: Automation

Browser integration requires Automation permissions:

**Setup:**
1. System Settings → Privacy & Security → Automation
2. Find SupaSidebar
3. Enable checkboxes for all browsers you use

**First Time:**
macOS will prompt when you first use shortcuts - click OK to grant access.

### Optional: Accessibility

Some browsers benefit from Accessibility permissions:

**System Settings → Privacy & Security → Accessibility** → Enable SupaSidebar

**Recommended for:**
- Comet (Perplexity) - Enhanced URL extraction
- Other browsers in edge cases

## Troubleshooting

### Smart Save Not Working

**Solutions:**
- Check Automation permissions for your browser
- Ensure browser window is active (frontmost)
- Verify browser is supported: [Browser Support](browserSupport)
- Try closing and reopening browser
- Restart SupaSidebar

### Permission Denied Error (-1743)

**Solutions:**
- Go to System Settings → Automation
- Enable SupaSidebar for the browser
- Restart both SupaSidebar and browser

### Wrong Browser Opens

**Solutions:**
- Right-click link → Set Browser → Choose correct browser
- Check Link Opening Behavior in Preferences → General
- See [Browser Selection](browserSelection) for details

### Live Tabs Not Showing

**Solutions:**
- Verify browser supports Live Tabs: [Browser Support](browserSupport)
- Check Automation permissions
- Enable Live Tabs: Preferences → Recents
- See [Live Tabs Troubleshooting](liveTabs#troubleshooting)
